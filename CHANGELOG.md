# Changelog

## [0.1.0] - 2025-12-06
### Added
- Initial public release of WisdomEye (简历解析、富化与评估管线)
- Core modules: 文本抽取、结构化解析、富化与评审、报告渲染
- Infra: LLM/Web 搜索适配、缓存、限流、重试、观测
- CLI 脚本：单文件与批处理流水线
- 基础测试覆盖与 CI 工作流
- 文档：README、架构说明与开源检查清单

### Security
- 提供 `.env.example` 与 `.gitignore` 以避免敏感信息泄露
